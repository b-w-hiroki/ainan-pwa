## プロジェクト概要
- プロジェクト名: AINAN 釣り×街おこし
- 方向性: PWAを前提に、触り心地の良い釣り体験と街おこし要素を組み合わせたサービス
- 前提: 振動なし、バックエンド含む全体設計、将来のアップデート/コンテンツ追加前提

## 現在の実装状況（事実ベース）
- フロントは Vite + React + TypeScript で起動可能
- PWA設定あり（`vite-plugin-pwa`）
  - `registerType: autoUpdate`
  - manifest: `name/short_name/display/theme_color/background_color/start_url` 設定済み
  - WorkboxのglobPatterns設定あり
- 画面実装（現状）
  - 長押しでキャストパワーを溜めるUI
  - 長押し時間からパワー（0〜100%）算出
  - パワーから飛距離（約5〜50m）算出して表示
  - 浮き位置が飛距離に応じて移動
  - 実装進捗カード（完了/未着手と達成率）を表示
- ビルドは成功済み（`npm run build`）

## 進捗カード上の現在ステータス

  - PWA土台（Vite + React）
  - キャストの長押し操作
